# About

Python implementation of "Challenge 2".

Requirement: private

# Usage

```shell
python3 main.py --input <input_directory>
```

input_directory can include subdirectories with csv files. Tree example:
```commandline
|-- LSE
|   |-- FLTR.csv
|   `-- GSK.csv
|-- NASDAQ
|   `-- TSLA.csv
`-- NYSE
    |-- ASH.csv
    `-- NMR.csv
```

Output will be in `output` directory
